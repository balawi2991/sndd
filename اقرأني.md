# 🌿 MintChat - منصة Chatbots ذكية

منصة SaaS متكاملة لإنشاء chatbots مدعومة بالذكاء الاصطناعي

---

## ✨ المميزات

- 🤖 **AI-Powered** - DeepSeek + RAG للردود الذكية
- 📚 **قاعدة معرفة** - رفع ملفات، روابط، نصوص
- 🎨 **قابل للتخصيص** - ويدجت جميل قابل للتخصيص بالكامل
- 💬 **محادثات** - عرض وإدارة جميع المحادثات
- 🔐 **آمن** - JWT authentication + bcrypt
- 📊 **لوحة تحكم** - إحصائيات وتحليلات

---

## 🚀 البدء السريع

### المتطلبات
- Node.js 20+
- PostgreSQL 14+ (أو حساب Railway)
- DeepSeek API key
- OpenAI API key

### النشر على Railway (10 دقائق)

#### 1. احصل على API Keys
- DeepSeek: https://platform.deepseek.com
- OpenAI: https://platform.openai.com

#### 2. أنشئ JWT Secret
```bash
openssl rand -base64 32
```

#### 3. ارفع على GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin YOUR_REPO_URL
git push -u origin main
```

#### 4. انشر على Railway
1. اذهب إلى https://railway.app/new
2. اختر "Deploy from GitHub repo"
3. اختر مستودعك
4. أضف PostgreSQL service
5. اضبط متغيرات البيئة
6. فعّل pgvector extension

**دليل مفصل:** [QUICK_START.md](./QUICK_START.md)

## 🧠 نظام RAG

### كيف يعمل؟
1. **الفهرسة:** محتوى → قطع → embeddings → pgvector
2. **الاسترجاع:** سؤال → بحث تشابه → أفضل 5 قطع
3. **التوليد:** سياق + سؤال → DeepSeek → رد + مصادر

## 🔌 API Endpoints

### المصادقة
- `POST /api/auth/signup`
- `POST /api/auth/signin`

### إدارة البوت
- `GET /api/bot`
- `PATCH /api/bot/:id`

### مواد التدريب
- `GET /api/training`
- `POST /api/training`
- `DELETE /api/training/:id`

### الدردشة
- `POST /api/chat`

### المحادثات
- `GET /api/conversations`
- `GET /api/conversations/:id`
- `DELETE /api/conversations/:id`
